# Frontend-Dev Repository

This repository is designed for the Web Development module. It is organized with topic-wise branches to help you learn and manage different concepts efficiently.

## Repository Structure

### Repository Name

**Frontend-Dev**

### Topic-wise Branches

1. **main** (empty or contains only `Readme.md`)
2. **intro-to-web-dev**
3. **html**
4. **html-block-&-inline-elements**
5. **html-iframes**
6. **css**
7. **flex**
8. **css-model**
9. **css-classes**
10. **responsive-css**
11. **bootstrap**

## Guidelines

- **Branching:**  
   Always create a new branch from the `main` branch for each topic listed above.
- **Work Isolation:**  
   Perform all your work within the respective topic branch.
- **No Merging:**  
   Do **not** merge any branch with the `main` branch.

## Getting Started

1. Clone the repository.
2. Checkout to the topic branch you want to work on.
3. Add your code and commit changes within that branch.
