Complete HTML Head Section

This page demonstrates all important elements that can be included in the <head> section of an HTML document.

Head Elements Included:

  • <title> - Document title shown in browser tab
  • <meta charset> - Character encoding declaration
  • <meta name="description"> - Page description for SEO
  • <meta name="author"> - Document author information
  • <meta name="viewport"> - Responsive design control
  • <meta name="keywords"> - Keywords for search engines
  • <base> - Base URL for relative links
  • <link> - Links to external resources (favicon, stylesheets)
  • <style> - Internal CSS styles
  • <script> - JavaScript code or references

Purpose of Each Element:

<title>
Defines the page title displayed in browser tabs and search results. Essential for SEO.
<meta charset>
Specifies character encoding (UTF-8 is standard for international support).
<meta name="description">
Provides a summary of the page content for search engines.
<meta name="author">
Identifies the document creator.
<meta name="viewport">
Controls page scaling and dimensions on mobile devices.
<base>
Sets a base URL that all relative URLs in the document use as their starting point.
<link>
Connects the document to external resources like stylesheets, fonts, or icons.
<style>
Contains CSS rules for styling the HTML elements on the page.
<script>
Contains or references JavaScript code for adding interactivity and dynamic behavior.
